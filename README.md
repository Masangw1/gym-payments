# Gym Payment Portal

A web-based payment entry system for gym management.

## Setup

1. Deploy the Google Apps Script as a web app
2. Copy the deployment URL
3. Replace `YOUR_GOOGLE_SCRIPT_URL_HERE` in index.html
4. Enable GitHub Pages in repository settings

## Features

- Single payment entry
- Batch entry for multiple payments
- Mobile-friendly interface
- Auto-generates Member IDs
